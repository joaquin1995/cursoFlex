Pseudoclases

Estilos especiales dependiendo del contexto, posicion o estado.

:root

La pseudo-clase de CSS :root selecciona el elemento raiz de un documento. Aplicando a HTML, :root representa el elemento &alt;html> y es identico al selector html, execpto que su nivel de especificidad es mayor. sirve para declarar varibales css.

:hover

La pseudo-clase de CSS :hover se presenta cuendo el usuario coloca el puntero sobre algun elemento, pero no necesariamente esta activo. En pantallas touch :hever es problematico.

:Active

La pseudo-clase de CSS :active cubre el caso en el que un elemento esta siendo activado por el usuario. cuendo se interactua con el raton, se corresponde al momento en el que el usuario presion el boton del raton hasta que el momento en que lo suelta. A menudo es usado en los elementos &alt;a&gc;y&alt;button>de HTML.

El orden si importa:

  • :link {...}
  • :visited {...}
  • :hover {...}
  • :active {...}

:target

La pseudo-clase de CSS :target representa el elemento unico, si existe alguno, con un id coincidente con el identificador de fragmentos de la URL del documento.

:nth-child :first-child :last-child

  • nth-child: selecciona el elemento indicaco que sea el enesimo hijo de su padre.
  • first-child: selecciona el elemento indicado que sea el primer hijo de su elemento padre.
  • last-child: selecciona el elemento indicado que sea el ultimo hijo de su elemento padre.

:nth-of-type :first-of-type :last-of-type

  • nth-of-type: selecciona el elemento indicado que sea el enesimo hijo de su tipo sin importar la jerarquia dentro del elemento padre.
  • first-of-type: selecciona el elemento indicado que sea el primer hijo de su tipo sin importar la jerarquia dentro del elemento padre.
  • last-of-type: selecciona el elemento indicado que sea el ultimio hijo de su tipo sin importar la jerarquia dentro del elemento padre.

:not

La pseudo-clase de CSS :not(X), es una notacion funcional que toma un selector simple X como argumento. coincidentecon un elemento que no esta representado por el argumento. X no debe contener otro selector de negacion.

:empty

La pseudo-clase de CSS :empty corresponde a un elemento sin ninguno nodo hijo. solo cuando hay nodos de elemento o texto con uno o mas carracteristicas dentro del elemento ese se convierte en vacio.
